SelfType 指的是方法所运行的类

Java 中并没有这个概念，在方法重载协变返回值可能会使用到。

子类实现clone（）方法时，需要强制转换cast类型

在Builder模式中如果子类扩展父类Builder,会发现编译不过

解决方式是，重载对应的方法修改返回类型，利用类型转换返回子类类型

另外一种解决方式是利用自身泛型类型PersonBuilder<T extends PersonBuilder>

调用上就是正常

自身泛型避免了类型转换，不需要使用cast
