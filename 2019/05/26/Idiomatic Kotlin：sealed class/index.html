bummock

被sealed标记的类是一个抽象类，并且构造函数是一个private的，所以并不能从外界访问，只能写在一个文件中。
同时生成了一个带DefaultConstructorMarker参数的构造方法，子类通过这个方法来初始化父类。不过这个类也是受保护的的类kotlin.jvm.internal。

sealed有一个好处是在编译期间，就可以知道所有的子类。在使用when表达式的时候，可以验证是否覆盖了所有条件。
